Predictive Drive Failure System

HDD/SSD의 SMART 데이터를 수집·분석해
고장을 사전에 예측하는 시스템

프로젝트 배경

🔴 문제

디스크 고장은 예고 없이 발생.
데이터 손실 전에 예측할 수 있는
방법이 필요.

💡 해결

디스크가 자체 기록하는
SMART 지표를 ML 모델로 분석해
고장 확률을 사전에 계산.

Kubernetes 배포 구조

Pod 구성

nginx Pod외부 트래픽 라우팅
backend PodFastAPI + ML 예측
frontend PodStreamlit 대시보드
postgres Pod데이터 영구 저장 (PVC)

운영 기능

  • ●  Pod 자동 재시작 (Deployment)
  • ●  영구 볼륨 (PersistentVolumeClaim)
  • ●  환경변수 ConfigMap / Secret 분리
  • ●  LoadBalancer로 외부 노출

minikube(로컬 k8s) 환경에서 4개 Pod 모두 Running 확인 완료. 실제 클라우드 서버(AWS EKS, GCP GKE) 배포 시 이미지 이름만 Docker Hub 주소로 교체하면 동일 매니페스트 사용 가능.
